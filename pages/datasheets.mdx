# Datasheets

Add a component datasheet PDF and Omen extracts its contents (pinouts, electrical specs, absolute ratings, etc.) so the assistant can reference it during conversations.

## Upload from the command line

The fastest way to upload a datasheet — works right from your terminal, no TUI needed:

```bash
# auto-detect the part number from the PDF
omen ~/Downloads/stm32f407.pdf

# or specify the part number explicitly
omen ~/Downloads/stm32f407.pdf STM32F407VGT6
```

The CLI uploads the file and returns immediately. It will be processed within a few minutes — no need to wait. Next time you open `omen` in a linked folder, the extracted content is synced automatically.

## Upload inside the TUI

Inside the interactive assistant, use the `/datasheet` slash command. You can upload directly:

```
/datasheet ~/Downloads/stm32f407.pdf STM32F407VGT6
```

Or open the widget for more control:

```
/datasheet
```

Press **`a`** to add. A single input appears that works two ways:

- **Upload a new PDF** — type or paste a file path (e.g. `~/Downloads/stm32f407.pdf`) and press Enter. You'll then be prompted for a part label like `STM32F407VGT6` (leave blank for auto-detect).
- **Attach an existing datasheet** — type a search term (e.g. `STM32F407`) and the list below filters to show matching datasheets already on your account. Press **`↓`** to move focus into the list, **`↑`** to go back to the input, and **Enter** on a listed datasheet to attach it to this folder/project instantly.

## Linked folders

When a folder is linked to a project (`/link <slug>`), datasheets are managed through the project on the web UI — the `/datasheet` upload command is disabled in the CLI. Use the web UI to add or remove datasheets for linked projects, then run `/refresh` to sync changes locally.

## Processing

After upload, the datasheet enters **processing** state while Omen OCRs and extracts sections from the PDF. This typically takes **2–5 minutes**. The widget updates the status live — no need to re-open it.

Once processing finishes, the extracted content is synced to `.omen/datasheets/<part-slug>/` on disk. For linked projects, this happens automatically whenever you open the assistant or run `/refresh`. For unlinked folders, sync happens on startup.

Need faster processing? Reach out at **d@usefirmware.com**.

## Remove a datasheet

Open the widget with `/datasheet`, navigate to the datasheet with **`↑`/`↓`**, and press **`r`** to remove it. Press **`esc`** to close the widget.

Removing a datasheet unlinks it from this folder/project — it stays on your account and can be re-attached later.
